Q.1 problem:     
  
  You are given a string (STR) of length N, consisting of only the lower case English alphabet.
  Your task is to remove all the duplicate occurrences of characters in the string.
   
   Input:
   abcadeecfb

   output:
   abcdef
  
var s='abcadeecfb';
let n=s.length;
let set=new Set();
for(let i=0;i < n;i++){
    set.add(s.charAt(i));
}
console.log(set);

 ouput:
 0: "a"

Q.2  Problem:
  You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps)
  
  input:
  abcadeecfb

  Ootput:
    a=2
    b=2
    c=2
    d=1
    e=2
    f=1

    const str = ["a","b","c","a","d","e","e","c","f","b"];

    const count = {};
   
    for (const i of str) {
      if (count[i]) {
        count[i] += 1;
      } else {
        count[i] = 1;
      }
    }
    console.log(count);
   

  output:
  a=2
  b=2
  c=2
  d=1
  e=2
  f=1
